# 🍦 Rosi Sorvetes — Dashboard Financeiro

Dashboard financeiro completo para sorveteria artesanal, construído com tecnologias modernas.

## 🚀 Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS v4** (estilização)
- **Recharts** (gráficos)
- **Lucide React** (ícones)
- **shadcn/ui** (componentes base)

## 📊 Funcionalidades

- **KPIs**: Receita mensal, despesas, lucro líquido e margem de lucro
- **Gráfico de Barras**: Receitas x Despesas (últimos 6 meses)
- **Tabela de Fornecedores**: nome, produto, valor, status do pagamento
- **Custos Fixos**: aluguel, energia, funcionários, insumos
- **Contas a Pagar**: com datas de vencimento e status
- **Sidebar** com navegação e logo
- **Responsivo** para mobile e desktop

## 🎨 Paleta de Cores

## 🛠️ Instalação

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```

## ☁️ Deploy (Vercel)

Conecte o repositório na Vercel — o `vercel.json` já está configurado.
